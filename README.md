# KK_Zayavka — калькулятор оценки проекта по критериям Комитета по культуре СПб

Интерактивный калькулятор оценки заявки на субсидию по официальным критериям
Комитета по культуре Санкт-Петербурга (Приложение 4 к распоряжению от 19.04.2026 № 440).

## Состав

- **`index.html`** — калькулятор оценки (один автономный файл, открывается в браузере, работает офлайн).
- **`критерии оценки.docx`** — первоисточник: полные формулировки критериев и значений баллов.

## Возможности калькулятора

- Полные официальные формулировки 4 критериев и подкритериев с примечаниями документа.
- Методика подсчёта: 4 критерия с весом 0,25; внутри — взвешенные подкритерии; итог 0–100.
- Предзаполнено консервативной экспертной оценкой проекта ПЕТЕРБУРГ.VR.
- Панель «Что подкрутить» — рычаги, отсортированные по приросту итоговых баллов.
- Экспертные карточки по каждому критерию (сильные стороны, доработки, риски).
- Автосохранение выбора в браузере (`localStorage`); кнопки «Сбросить» и «Печать / PDF».
